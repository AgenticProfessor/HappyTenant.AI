# Happy Tenant - Tenant Portal Design

## Design Philosophy for Tenants

The tenant portal should feel like a **helpful concierge** - simple, accessible, and reassuring. Tenants interact less frequently than landlords, so every interaction must be:
- Immediately clear (no learning curve)
- Task-focused (get in, get out)
- Mobile-optimized (most access via phone)
- Anxiety-reducing (payments, maintenance can be stressful)
