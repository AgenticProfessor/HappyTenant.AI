# Happy Tenant - Website Design

## Design Direction: Modern & Warm

### Visual Identity

**Color Palette**
```
Primary:      #4F46E5 (Indigo) - Trust, professionalism
Secondary:    #10B981 (Emerald) - Growth, success, money
Accent:       #F59E0B (Amber) - Warmth, attention
Background:   #FAFAF9 (Warm white)
Surface:      #FFFFFF
Text Primary: #1C1917 (Warm black)
Text Secondary: #78716C (Warm gray)
```

**Typography**
```
Headlines:    Inter (Bold/Semibold) - Clean, modern
Body:         Inter (Regular) - Readable, friendly
Accent:       Outfit (Medium) - For CTAs and highlights
```

**Design Principles**
- Rounded corners (12-16px) for warmth
- Soft shadows instead of hard borders
- Subtle gradients for depth
- Generous white space
- Warm undertones throughout
- Micro-animations for delight
